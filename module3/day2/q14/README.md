# Enhanced Todo List Application

This is an advanced Todo List web app built using HTML, CSS, and JavaScript.  
It supports task creation, deletion, search, and persistent storage using localStorage.

## Features

### 1. Add Tasks
- Enter a task in the input field.
- Click "Add Task" to save it.
- Each task is stored as an object with:
  - Unique ID
  - Task text
  - Completion status

### 2. Mark Task as Completed
- Click on any task to toggle completion.
- Completed tasks appear with a line-through.

### 3. Delete Tasks
- Click the "Delete" button beside any task to remove it permanently.

### 4. Search Tasks in Real-Time
- Type in the search bar to filter tasks instantly.

### 5. LocalStorage Support
- All tasks are saved in `localStorage`.
- Tasks stay available even after reloading the page.
